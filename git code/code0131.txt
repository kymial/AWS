clear : 터미널 화면 청소
pwd : 터미널의 현재 경로(위치)를표시
ls: 현재 폴더의 파일 혹은 폴더리스트를 보여줌
	- : 옵션 추가 명령어
		a : 모든 파일 및 폴더를 보여준다.
		l : 파일 혹은 폴더의 상세정보를 보여준다.
			- / rwx (1) / rwx (2) / rwx (3)
			- d는 폴더, l 바로가기, - 파일
			- 1  rwx : 소유자에 대한 권한 
			- 2. rwx : 소유그룹에 대한 권한
			- 3. rwx : 기타 타인
			- r : 읽기 권한 w : 작성 권한 x : 실행 권한
cd : 디렉토리 폴더 이동
/ : 현재 위치
. : 뒤
